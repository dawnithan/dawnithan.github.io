My Projects

Mobile Applications

As part of Building City Dashboards, I created and published two mobile applications which used free and open-source software and libraries to make practical use of open data. Cork Live Transport used information regarding the car parks and bicycle stations in Cork City to show their availabilty to the public. Ballyfermot Community Services Map used data gathered by students of St. Dominics Secondary School in Ballyfermot to show the location, website, and contact details of various different services within Ballyfermot, as a 70th year anniversary celebration.

Web APIs

I've used the Twitter API to create a Discord bot that will take in a tweet and output the images within it that aren't shown in the Discord tweet embed. The purpose of making this tool was to circumvent the lack of Discord showing all of the tweet's images, and was a fun journey through making a bot and using the Twitter API. Additionally, I've also made a train timetable which connects to the Irishrail API and returns details for a few select stations, as proof of concept.
